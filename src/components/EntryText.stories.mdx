# Text Entry

This component renders a text input with a label and programmable hint text.

## Usage

```html
<entry-text
  label="Email"
  for="email"
/>
```

export const Template = (args) => ({
  components: { Component },
  setup() {
    return { args }
  },
  template: `<Component v-bind="args" />`,
})
